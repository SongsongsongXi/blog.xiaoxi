## 关于本站

这是一个使用 Modern Markdown Blog 模板搭建的博客。

您可以在这里分享您的技术文章、学习笔记和生活感悟。

### 功能特色

-   **轻量快速**：基于 FastAPI 和原生 JavaScript 构建，无需繁重的框架。
-   **Markdown 驱动**：所有文章均使用 Markdown 编写，专注内容创作。
-   **功能丰富**：支持代码高亮、数学公式、图表、音乐播放器等。

### 开源模板

本项目已开源，您可以访问 [GitHub 仓库](https://github.com/SongsongsongXi/blog.xiaoxi) 获取模板，并根据 `README.md` 文件中的指引进行部署。
